INSTALLATION STEPS
================================================================================

1. CREATE DATABASE:
   mysql -u root -p -e "CREATE DATABASE web_krs CHARACTER SET utf8mb4"

2. IMPORT SCHEMA:
   mysql -u root -p web_krs < database/schema.sql

3. IMPORT SEED DATA:
   mysql -u root -p web_krs < database/seed.sql

4. VERIFY CONFIG:
   Edit includes/config.php if using different credentials

5. CREATE UPLOAD DIR:
   mkdir -p public/assets/uploads
   chmod 755 public/assets/uploads

6. RUN APPLICATION:
   Option A: XAMPP - Copy to htdocs/
   Option B: PHP Server - php -S localhost:8000 -t public

7. TEST LOGIN:
   URL: http://localhost:8000 (or configured URL)
   Use admin/password123

GRADE CONVERSION REFERENCE
================================================================================

LETTER GRADE MAPPING:
  A     ≥ 85.00   Weight: 4.0  (Excellent)
  B+    70-84.99  Weight: 3.5  (Very Good)
  B     60-69.99  Weight: 3.0  (Good)
  C+    55-59.99  Weight: 2.5  (Satisfactory)
  C     50-54.99  Weight: 2.0  (Adequate)
  D     40-49.99  Weight: 1.0  (Poor)
  E     < 40.00   Weight: 0.0  (Fail)

GRADE CALCULATION:
  Nilai Akhir = 0.2×Tugas + 0.3×UTS + 0.5×UAS

SKS LIMITS BY IPK:
  IPK ≥ 3.50    → 24 SKS
  3.00-3.49     → 22 SKS
  2.50-2.99     → 20 SKS
  2.00-2.49     → 18 SKS
  < 2.00        → 15 SKS

GPA CALCULATION:
  IPS = Σ(Grade Weight × SKS) / Σ(SKS)  [per semester]
  IPK = Σ(Grade Weight × SKS) / Σ(SKS)  [cumulative]
